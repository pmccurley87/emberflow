# Demo Project

A minimal Emberflow consumer project: its own node (`TriageOrder`) and its own flow (`order-triage`), used both as a worked example and as this repo's default `npm run dev:project` target.

Run `npm run dev:project` from the repo root to boot the runner against this project.

Then try `npm run cli -- run order-triage --scenario vip` to execute it end-to-end.
